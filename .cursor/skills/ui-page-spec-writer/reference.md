# 参考：完整输出示例

以下示例展示了对"华为云控制台工作台页面"的设计文档输出，可作为文档质量标杆。

---

## 输入

用户需求："设计一个华为云控制台的工作台页面，包含顶栏导航、左侧收藏项目栏、中间展示欢迎语+最近访问+所有项目卡片网格、右侧展示公告/活动/帮助文档。"

## 输出

```markdown
# 华为云控制台工作台页面 — 设计需求文档

本文档描述华为云控制台工作台（Dashboard）页面的详细设计需求，
用于指导 Pencil 设计工具基于组件库
`designsystem/devUI/components/devUI2.pen` 进行设计实现。

---

## 1. 整体布局与视觉风格

**布局结构**：经典四区布局——顶部全局导航栏、左侧固定侧边导航、
中间主内容区、右侧辅助信息栏。纵向为顶栏单行、主体左右分栏
（左窄右宽，中间主内容区最宽）。

**视觉风格**：
- 色调：科技感蓝白系，背景浅灰 #EEF0F5，卡片白底 #FFFFFF，
  主色蓝 #526ECC
- 组件特征：圆角 8px 卡片、轻微阴影区分层级、留白充足
- 文字：主文字 #252B3A、次文字 #575D6C、标签浅绿底+深灰字

**画布尺寸**：1920×1080

---

## 2. 核心区域详解

### A. 顶部全局导航栏

- **左侧**：汉堡菜单图标 + 华为云 Logo + 文本导航项
  （控制台、华北-北京四▾、首页、工作台✓、效能洞察、
  服务▾、华为开源镜像站）
- **右侧**：通知铃铛图标 + 帮助问号图标 + 语言切换（EN / HZ✓）
- 「工作台」为选中态高亮

**还原要点**：全宽单行；左Logo+导航，右图标+语言切换；
选中项蓝色下划线。

---

### B. 左侧侧边导航栏

- 白底固定宽度 240px，垂直列表
- 标题「我的关注」+ 星形图标
- 导航项：默认项目（文件夹图标）、API服务创建的项目（API图标）、
  已归档（归档箱图标）

**还原要点**：白底竖排，图标+文字，支持选中/悬浮状态区分。

---

### C. 主内容区

#### C.1 欢迎与最近访问

- 大号欢迎语「Hello, Jingwen」
- 标题「最近访问」+ 3 张横向卡片（彩色图标+项目名称）
- 右侧蓝色科技感 3D 插画装饰

**还原要点**：欢迎语 24px 粗体；3 卡片等宽横排 gap=16。

#### C.2 所有项目

- 标题「所有项目」
- 筛选栏：页签（全部类型✓/项目群/IPD/Scrum/看板）+
  新建按钮 + 下拉选择 + 搜索框 + 视图切换（卡片✓/列表）
- 项目卡片网格：3 列，每卡含字母图标+收藏+更多+
  标题+标签+创建者信息

**还原要点**：筛选栏同行排列；卡片 3 列 gap=16；
单卡包含图标/标题/标签/底部元信息。

---

### D. 右侧辅助信息栏

- 宽度约 384px，白底圆角卡片
- **公告**：标题+品牌升级文案+时间「2天前」
- **活动**：标题「频繁重新登录?」+说明文案+插画
- **帮助文档**：标题+「查看更多」+ 3×3 图标文字网格

**还原要点**：三块独立卡片纵向排列 gap=16。

---

## 3. 组件使用与还原要点

| 区域 | 建议组件 | 说明 |
|------|----------|------|
| 顶栏 | 顶部导航栏 `Rpv7B` | 改菜单文案和右侧图标 |
| 左侧栏 | 左侧选择菜单卡片 `6aBNN` | 改标题和导航项文案 |
| 页签筛选 | 页签Pills `Mtrw0` | 5 项，改文案 |
| 新建按钮 | 按钮-主要 `RoYUx` | 改文案为「+ 新建」 |
| 搜索 | 搜索框 `X93Mt` | 改占位符 |
| 项目卡片 | 项目卡片 `LcKkl` | 改标题/标签/元信息 |
| 公告/活动 | 公告卡片 `3kAYs` / 活动卡片 `ioZGG` | 改文案和图片 |
| 帮助文档 | 基本卡片 `YCOKr` | 内插 3×3 图标网格 |

---

## 4. 实现顺序建议

1. 创建画布 1920×1080，设背景色 #EEF0F5
2. 搭建顶栏（全宽，复制顶部导航栏组件）
3. 搭建左侧栏（固定宽度 240px）
4. 搭建主内容区 + 右侧栏分栏
5. 填充 C.1 欢迎与最近访问区
6. 填充 C.2 筛选栏 + 项目卡片网格
7. 填充 D 右侧三块卡片
8. 微调间距、图标、状态
```

## 常见页面类型拆解模式

### 数据列表页

```
区块拆解：
├── A. 顶部导航栏
├── B. 侧边导航栏
├── C. 头信息（标题+面包屑+操作按钮）
├── D. 筛选/搜索工具栏
│   ├── D.1 筛选器组
│   ├── D.2 搜索框
│   └── D.3 操作按钮（新建/导出）
├── E. 数据表格
│   ├── E.1 表头（列标题+排序+全选）
│   └── E.2 表体（行数据+操作列）
└── F. 分页器
```

### 表单配置页

```
区块拆解：
├── A. 顶部导航栏
├── B. 面包屑导航
├── C. 页面头信息（标题+描述）
├── D. 表单区域
│   ├── D.1 基础信息分组（输入框×N）
│   ├── D.2 高级配置分组（下拉选择+开关+复选）
│   └── D.3 操作按钮栏（提交+取消）
└── E. 辅助提示信息
```

### 仪表盘/概览页

```
区块拆解：
├── A. 顶部导航栏
├── B. 侧边导航栏
├── C. 头信息（欢迎语/概览标题）
├── D. 统计指标卡片行（3-4 张指标卡）
├── E. 内容区
│   ├── E.1 图表/数据可视化区
│   ├── E.2 最近活动/动态列表
│   └── E.3 快捷操作入口
└── F. 辅助信息侧栏（公告/帮助）
```

### 详情页

```
区块拆解：
├── A. 顶部导航栏
├── B. 面包屑导航
├── C. 头信息（标题+状态标签+操作按钮）
├── D. 页签切换栏
├── E. 详情内容区（根据选中页签）
│   ├── E.1 基本信息（键值对列表）
│   ├── E.2 关联数据（子表格/卡片列表）
│   └── E.3 操作日志/变更记录
└── F. 底部操作栏（可选）
```
