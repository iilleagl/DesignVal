# Web 产品首页 — 设计需求文档

本文档描述 B 端 Web 产品首页的详细设计需求，用于指导 Pencil 设计工具基于组件库 `designsystem/devUI/components/devUI2.pen` 进行设计实现。风格：极简现代、扁平化、B 端产品。

---

## 1. 整体布局与视觉风格

**布局结构**：顶栏（全宽）+ 下方内容区（左侧手风琴 + 中心主内容 + 右侧功能特性区）。经典三栏仪表盘型。

**视觉风格**：
- **色彩方案**：主色黑色 #191919（按钮、标题、图标）；辅助色蓝色 #0A59F7（超链接、文字按钮）；中性色白 #FFFFFF、背景浅灰 #F5F5F5；整体留白充足，细节清晰。
- **组件特征**：扁平化、极简现代、卡片带阴影。
- **文字**：Inter 无衬线字体，主/次文字清晰。

**画布尺寸**：1920×1080（或 1440×900）

---

## 2. 核心区域详解

### A. 顶部导航栏

- 位置：页面最顶部，全宽，固定高度约 56px；背景纯净白色 #FFFFFF，文字黑色；选中项变为主题背景色（蓝色系）。
- 布局：最左侧为**侧边导航收起图标** → **华为品牌 Logo** → 核心功能入口（**控制台、区域切换下拉、首页、工作台、效能看板、服务**，图标+文字按钮）→ 右侧**用户中心**（消息通知、待办中心、中英文切换、头像）。

**内容物料**：

| 元素 | 类型 | 具体内容 |
|------|------|----------|
| 收起图标 | 图标 | 侧边导航收起 |
| Logo | 文本/图标 | 华为（Huawei） |
| 菜单项1 | 文本+图标 | 控制台 |
| 菜单项2 | 文本+图标 | 区域切换（下拉） |
| 菜单项3 | 文本+图标 | 首页（当前选中） |
| 菜单项4 | 文本+图标 | 工作台 |
| 菜单项5 | 文本+图标 | 效能看板 |
| 菜单项6 | 文本+图标 | 服务 |
| 右侧-消息 | 图标 | 消息通知 |
| 右侧-待办 | 图标 | 待办中心 |
| 右侧-语言 | 图标/文本 | 中英文切换 |
| 右侧-用户 | 文本/头像 | 用户头像 |

**还原要点**：使用顶部导航栏组件 Uexy2，descendants 填齐上述文案；左侧增加侧边栏收起图标。

---

### B. 内容区背景与装饰

- 位置：顶栏下方整块内容区。
- 背景：浅灰色渐变 + 抽象科技元素（代码符号、3D 修饰图标），不干扰主体内容。

**还原要点**：主容器背景 #F5F5F5 或浅灰渐变；可加低透明度装饰图层。

---

### C. 左侧手风琴导航

- 位置：内容区左侧，固定宽度约 200–240px。
- 内容：**我的关注、所有项目、已归档**；默认选中「所有项目」；样式为图标+文字。

**内容物料**：

| 元素 | 类型 | 具体内容 |
|------|------|----------|
| 项1 | 文本+图标 | 我的关注 |
| 项2 | 文本+图标 | 所有项目（默认选中） |
| 项3 | 文本+图标 | 已归档 |

**还原要点**：使用手风琴侧边导航 K0pVg（或 kyl45），descendants 填齐三项文案。

---

### D. 中心内容区 — 主标题与最近访问

- 主标题：**Hello，Liutinting**（用户名称），加粗大号字体。
- 副标题：**最近访问**，展示最近访问的项目卡片，卡片包含图标+标题。

**内容物料**：

| 元素 | 类型 | 具体内容 |
|------|------|----------|
| 主标题 | 文本 | Hello，Liutinting |
| 副标题 | 文本 | 最近访问 |
| 最近访问卡片×N | 卡片 | 图标 + 项目标题（示例：项目A、项目B、项目C） |

**还原要点**：主标题大号加粗；下方横向排列最近访问项目卡片（2–4 个）。

---

### E. 中心内容区 — 所有项目

- 功能：Tab 切换项目类型、新建项目、下拉筛选分组、搜索项目、切换列表及卡片展示形式。
- 卡片：每行 3–4 个并列项目卡片；每个卡片包含图标、项目标题、类型、创建人、收藏功能、设置功能；卡片有阴影效果。

**内容物料**：

| 元素 | 类型 | 具体内容 |
|------|------|----------|
| Tab1 | 文本 | 全部 |
| Tab2 | 文本 | 我创建的 |
| Tab3 | 文本 | 我参与的 |
| 新建项目 | 按钮 | 新建项目 |
| 筛选 | 下拉 | 分组筛选 |
| 搜索框 | 占位 | 搜索项目 |
| 视图切换 | 图标 | 列表 / 卡片 |
| 项目卡片×N | 卡片 | 图标、项目标题、类型、创建人、收藏、设置（示例 6–8 个） |

**还原要点**：页签 + 头信息（新建+筛选+搜索）+ 项目卡片网格；使用项目卡片组件 9qbEz 并填文案。

---

### F. 中心内容区 — 新特性展播

- 形式：以**步骤条**形式展示产品特性功能，支持关闭。

**内容物料**：

| 元素 | 类型 | 具体内容 |
|------|------|----------|
| 标题 | 文本 | 新特性展播 |
| 步骤1 | 文本 | 新特性一标题与简述 |
| 步骤2 | 文本 | 新特性二标题与简述 |
| 步骤3 | 文本 | 新特性三标题与简述 |
| 关闭 | 图标/按钮 | 关闭 |

**还原要点**：使用步骤条组件 Zavv1 或 qPPvF，可顶部放「新特性展播」标题与关闭按钮。

---

### G. 右侧功能特性区

分卡片展示四大模块，固定宽度约 280–320px。

**G1. 公告**：支持轮播。  
**内容物料**：公告标题、公告条目 1、2（标题+摘要）。

**G2. 活动**：Banner 插画形式展示热门活动，支持滚动。  
**内容物料**：活动标题、活动 Banner 图/标题（1–2 条）。

**G3. 帮助文档**：产品介绍、快速入门、用户指南、最佳实践等，图标+文字。  

| 元素 | 类型 | 具体内容 |
|------|------|----------|
| 模块标题 | 文本 | 帮助文档 |
| 链接1 | 文本+图标 | 产品介绍 |
| 链接2 | 文本+图标 | 快速入门 |
| 链接3 | 文本+图标 | 用户指南 |
| 链接4 | 文本+图标 | 最佳实践 |

**G4. 版本历程**：采用**竖向时间轴**形式展示版本变更。  
**内容物料**：版本历程标题、版本 1（版本号+说明）、版本 2、版本 3。

**还原要点**：四块卡片上下排列；可使用基本卡片、步骤条纵向 FYkld 等。

## 3. 组件使用与还原要点

| 区域 | 建议组件 | 说明 |
|------|----------|------|
| 顶部导航栏 | 顶部导航栏 Uexy2 | 填 Logo、菜单项、用户区文案；左侧加收起图标 |
| 左侧手风琴 | 手风琴侧边 K0pVg | 三项：我的关注、所有项目、已归档 |
| 主标题 | 自建 text | 大号加粗 Hello，Liutinting |
| 最近访问 | 项目卡片 9qbEz | 2–4 个卡片，填图标+标题 |
| 所有项目区 | 页签 + 头信息 + 搜索 + 项目卡片 | 页签 nCtvQ/Mtrw0；搜索 pClmN；项目卡片 9qbEz |
| 新特性展播 | 步骤条 Zavv1/qPPvF | 横向步骤条 + 关闭按钮 |
| 右侧公告/活动/帮助/版本 | 基本卡片 7ipso + 自建内容 | 四块卡片，版本用纵向步骤条 FYkld |

**设计规范**：主色 #191919、辅助色 #0A59F7、背景 #F5F5F5、白 #FFFFFF；字号 12/14/16/20/24/36 px；间距 4 的倍数；Inter 字体。

---

## 4. 实现顺序建议

1. 搭整体框架：画布 → 顶栏 → 内容区三栏（左中右）。
2. 顶栏：插入顶栏 ref + 填 descendants（Logo、菜单、用户中心），左侧加收起图标。
3. 左侧：插入手风琴 ref + 填三项文案。
4. 中心：主标题 → 最近访问卡片 → 所有项目（Tab+头信息+搜索+项目卡片网格）→ 新特性步骤条。
5. 右侧：公告卡片 → 活动卡片 → 帮助文档卡片 → 版本历程时间轴。
6. 背景与装饰：内容区背景色、可选科技元素。
7. 细节：阴影、圆角、选中态颜色。
