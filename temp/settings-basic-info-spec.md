# 设置-基本信息 — 设计需求文档

本文档描述「设置-基本信息」页面的详细设计需求，用于指导 Pencil 设计工具
基于组件库 `designsystem/devUI/components/devUI2.pen` 进行设计实现。遵循华为云 CodeArts 设计规范，明亮浅色底、圆角规范、高保真。

---

## 1. 整体布局与视觉风格

**布局结构**：顶栏（固定高度 48px）+ 左侧边栏（固定宽度 192px）+ 主内容区（右侧剩余区域）。左侧栏含标题、搜索、通用配置分组，其中「基本信息」为当前/悬浮项；主内容区为基本信息表单与删除组织区块。

**视觉风格**：
- 色调：明亮浅色底，主按钮色 #0A59F7，主要文字 #191919，次要文字 #595959；删除按钮为红色。
- 组件特征：华为云 CodeArts 风格，圆角与 DevUI 规范一致（如 radius-sm 4px、radius-lg 8px），卡片/输入框带适当圆角与分割。
- 文字：主文字 #191919、次要/辅助 #595959；字体 Noto Sans SC / HarmonyOS Sans SC。

**画布尺寸**：1440×900（或 1920×1080，以实际主内容区舒适阅读为准）。

---

## 2. 核心区域详解

### A. 顶部全局导航栏

- 位置：页面最顶部，全宽。
- 尺寸：高度 48px，宽度填满。
- 布局：水平排列，左侧可放 Logo/产品名，右侧可放通知、帮助、用户信息等。
- 视觉：浅底、底部分割线，与主内容区分隔。

**内容物料**：

| 元素 | 类型 | 具体内容 |
|------|------|----------|
| 产品/系统名称 | 文本 | CodeArts 或 设置（按业务定） |
| 右侧区域 | 占位 | 通知、帮助、用户头像等（可简化） |

**还原要点**：使用组件库「顶部导航栏」ref，高度设为 48px；替换文案与图标。

---

### B. 左侧导航栏

- 位置：顶栏下方、页面左侧，宽度 192px。
- 尺寸：宽 192px，高度填满剩余视口。
- 布局：自上而下依次为：导航标题、搜索框、通用配置分组（含「基本信息」菜单项）；鼠标在「基本信息」上时该项高亮。
- 视觉：浅灰或白色背景，与主内容区有分割线；选中/悬浮态明确。

**内容物料**：

| 元素 | 类型 | 具体内容 |
|------|------|----------|
| 导航标题 | 文本 | 设置（或 系统设置） |
| 搜索框占位 | 文本 | 搜索菜单/配置项 |
| 分组标题 | 文本 | 通用配置 |
| 菜单项-基本信息 | 文本 | 基本信息（当前项/悬浮高亮） |
| 其他菜单项 | 文本 | 可按需增加占位，如 安全设置 |

**还原要点**：使用侧边导航/手风琴类组件，宽度固定 192px；搜索用搜索框组件；确保「基本信息」有选中或悬浮样式。

---

### C. 主内容区 - 区块标题

- 位置：顶栏下、左侧栏右侧，内容区顶部。
- 布局：单行标题，可选面包屑在上方。
- 内容：页面级标题「基本信息」。

**内容物料**：

| 元素 | 类型 | 具体内容 |
|------|------|----------|
| 页面标题 | 文本 | 基本信息 |

**还原要点**：字号 16px 或 18px，字色 #191919，与下方表单区留足间距。

---

### D. 主内容区 - 组织信息卡片

- 位置：标题下方，主内容区上半部分。
- 布局：卡片内从上到下：图标设置行、组织名称、组织域名、区域、组织描述、操作按钮行（保存、取消）。
- 图标设置：支持「设置」入口；编辑图标时展示可配图标颜色（色板或取色）及预制图标列表可选。
- 主按钮（保存）：填充色 #0A59F7；取消为次要按钮。
- 视觉：卡片圆角、浅底，表单项标签与输入框分明；主文字 #191919，占位/辅助 #595959。

**内容物料**：

| 元素 | 类型 | 具体内容 |
|------|------|----------|
| 卡片标题/区块标题 | 文本 | 组织信息（可选） |
| 图标标签 | 文本 | 组织图标 |
| 设置图标/按钮 | 文本/图标 | 设置（或齿轮图标） |
| 编辑图标时-颜色标签 | 文本 | 图标颜色 |
| 编辑图标时-预制图标 | 图标列表 | 若干预制图标可选 |
| 组织名称-标签 | 文本 | 组织名称 |
| 组织名称-占位 | 文本 | 请输入组织名称 |
| 组织域名-标签 | 文本 | 组织域名 |
| 组织域名-占位 | 文本 | 请输入组织域名 |
| 区域-标签 | 文本 | 区域 |
| 区域-占位 | 文本 | 请选择区域 |
| 组织描述-标签 | 文本 | 组织描述 |
| 组织描述-占位 | 文本 | 请输入组织描述（可选） |
| 主按钮 | 文本 | 保存 |
| 次按钮 | 文本 | 取消 |

**还原要点**：主按钮 fill #0A59F7；输入框、下拉用 DevUI 表单项组件；图标编辑区可用一组预设图标 + 颜色选择器表示。

---

### E. 主内容区 - 删除组织区块

- 位置：组织信息卡片下方，主内容区内。
- 布局：删除预警提示文案 +「删除组织」按钮；按钮为红色（危险按钮样式）。
- 视觉：预警文字可用次要色 #595959；按钮红色醒目。

**内容物料**：

| 元素 | 类型 | 具体内容 |
|------|------|----------|
| 预警说明 | 文本 | 删除组织后，该组织下的所有项目与数据将被永久删除，且无法恢复。请谨慎操作。 |
| 删除按钮 | 文本 | 删除组织 |

**还原要点**：按钮使用危险/红色样式；主文字 #191919，说明文字 #595959。

## 3. 组件使用与还原要点

实现时优先从 `designsystem/devUI/components/devUI2.pen` 选用组件，复制到设计稿后通过 ref 引用；引用后**必须**修改实例内的文案、图标、占位内容以符合本页业务语义。

| 区域 | 建议组件 | 说明 |
|------|----------|------|
| 顶部导航栏 | 顶部导航栏 (如 Rpv7B) | 高度改为 48px，替换 Logo/菜单文案 |
| 左侧导航 | 侧边导航/手风琴 (如 kyl45、JGEqF) | 宽 192px，标题「设置」，分组「通用配置」，菜单项「基本信息」 |
| 搜索 | 搜索框 (如 X93Mt) | 放在侧栏标题下方 |
| 页面标题 | 无匹配则自建 | 文本「基本信息」，16/18px，#191919 |
| 组织信息表单 | 表单项-文本输入框 (Laiht)、表单项-下拉 (qw7Ux) | 组织名称、域名、区域、描述；主按钮 #0A59F7 |
| 保存/取消 | 按钮-主要 (RoYUx)、按钮-次要 (qTc8e) | 保存主色 #0A59F7，取消次要 |
| 删除组织 | 按钮-危险/次要 + 文本 | 红色按钮，预警文案 #595959 |
| 卡片容器 | 基本卡片 (7ipso) | 包裹组织信息与删除区块 |

**设计规范**：主按钮色 #0A59F7；主要文字 #191919，次要文字 #595959；间距 4 的倍数；圆角与组件库一致；华为云 CodeArts 风格、高保真。

---

## 4. 实现顺序建议

1. 搭整体框架：画布 → 顶栏 48px + 左侧栏 192px + 主内容区。
2. 顶栏：插入顶部导航组件，改高度与文案。
3. 左侧栏：插入侧边导航 + 搜索框，填标题「设置」、分组「通用配置」、菜单项「基本信息」并做选中/悬浮态。
4. 主内容区：页面标题「基本信息」。
5. 组织信息卡片：卡片内图标设置、四个表单项、保存/取消按钮；主按钮 #0A59F7。
6. 删除组织区块：预警文案 + 红色删除按钮。
7. 细节：圆角、间距、颜色 token 与截图验证。
