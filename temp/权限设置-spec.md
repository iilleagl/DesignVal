# 组织权限-权限设置 — 设计需求文档

本文档描述「组织权限-权限设置」页面的详细设计需求，用于指导 Pencil 设计工具基于组件库 `designsystem/devUI/components/devUI2.pen` 进行设计实现。

---

## 1. 整体布局与视觉风格

**布局结构**：顶部导航栏（高 48px）+ 下方三栏布局：左侧导航（宽 192px）+ 中间主内容区 + 右侧角色导航（宽 192px）。主内容区为角色权限设置区域。

**视觉风格**：
- 色调：明亮浅色底，主文字色 #191919，次要文字色 #595959，勾选框/选中色 #0A59F7
- 组件特征：华为云 CodeArts 设计规范，圆角规范（radius-sm 4px、radius-md 6px、radius-lg 8px），高保真
- 文字：Noto Sans SC，主文字 14px/16px，#191919；次要文字 12px/14px，#595959

**画布尺寸**：1920×1080（或 1440×900）

---

## 2. 核心区域详解

### A. 顶部全局导航栏

- 位置：页面最顶部，全宽，固定高度 48px
- 背景：白色 #FFFFFF
- 布局：水平排列，左侧 Logo/产品名，右侧用户/通知等
- 交互：标准顶部导航交互

**内容物料**：

| 元素 | 类型 | 具体内容 |
|------|------|----------|
| Logo | 文本/图标 | CodeArts |
| 菜单项1 | 文本 | 首页 |
| 菜单项2 | 文本 | 项目 |
| 菜单项3 | 文本 | 服务 |
| 右侧-通知 | 图标 | 消息通知 |
| 右侧-用户 | 文本/头像 | 用户头像 |

**还原要点**：使用顶部导航栏组件 ref，高度 48px。

---

### B. 左侧导航栏

- 位置：顶栏下方左侧，固定宽度 192px
- 背景：浅灰 #F5F5F5 或白色
- 布局：垂直列表，顶部有导航标题，含搜索框
- 内容：通用配置、权限管理（当前选中/hover）、其他菜单项
- 交互：鼠标在「权限管理」项时高亮

**内容物料**：

| 元素 | 类型 | 具体内容 |
|------|------|----------|
| 导航标题 | 文本 | 组织管理 |
| 搜索框 | 占位 | 搜索菜单 |
| 菜单项1 | 文本+图标 | 通用配置 |
| 菜单项2 | 文本+图标 | 权限管理（当前选中） |
| 菜单项3 | 文本+图标 | 成员管理 |
| 菜单项4 | 文本+图标 | 其他 |

**还原要点**：使用工具链侧边栏或手风琴侧边导航，宽度 192px，descendants 填齐文案。

---

### C. 右侧角色导航（树导航）

- 位置：主内容区右侧，固定宽度 192px
- 背景：白色或浅灰
- 布局：树形结构，展示角色列表
- 内容：角色层级树，如 管理员、开发者、访客 等

**内容物料**：

| 元素 | 类型 | 具体内容 |

| 区域标题 | 文本 | 角色 |
| 树节点1 | 文本 | 管理员 |
| 树节点2 | 文本 | 开发者 |
| 树节点3 | 文本 | 访客 |
| 树节点4 | 文本 | 自定义角色 |

**还原要点**：使用树形导航组件，宽度 192px。

---

### D. 角色权限设置区域 — 头部

- 位置：左侧导航与右侧角色导航之间的主内容区顶部
- 布局：标题 + 页签（权限、成员）+ 搜索框

- 布局：垂直列表，展示华为云各服务名称
- 内容：Req、Pipeline、代码托管、制品仓库、部署、测试等

**内容物料**：

| 服务项7 | 文本 | 项目管理 |

**还原要点**：左侧选择菜单或列表，选中项高亮。

---

### F. 权限页签 — 右侧服务卡片区

- 位置：主内容区右侧，与左侧服务列表对应
- 布局：宽版卡片，上下组合，每个服务一张卡片
- 卡片结构：图标 + 服务名 + 关键任务列表，每个任务有 创建、只读、设置、删除 四个权限勾选框
- 卡片有全选功能（全选该服务下所有权限）
- 勾选框颜色：#0A59F7
- 主要文字色：#191919
- 次要文字色：#595959

**内容物料**（以 Req 为例）：

| 任务1-只读 | 复选框 | 只读 |

| 任务1-删除 | 复选框 | 删除 |
| 任务2-名称 | 文本 | 迭代计划 |
| 任务2-创建/只读/设置/删除 | 复选框 | 同上 |

| 元素 | 类型 | 具体内容 |

## 3. 组件使用与还原要点

| 区域 | 建议组件 | 说明 |
|------|----------|------|
| 顶部导航栏 | 顶部导航栏 Uexy2 | 高度改为 48px |
| 左侧导航 | 工具链侧边栏 E36oP 或 手风琴 K0pVg | 宽度 192px，填齐菜单文案 |
| 搜索框（左侧导航） | pClmN | 占位「搜索菜单」 |
| 右侧角色树 | 树形结构/左侧选择菜单 VP3uY | 宽度 192px |
| 头信息+页签 | TUWxd + 页签Pills Z70QE | 权限/成员两个页签 |
| 搜索框（主区） | pClmN | 占位「搜索服务或权限」 |
| 左侧服务列表 | 左侧选择菜单 VP3uY 或自建 | 服务名称列表 |
| 服务卡片 | 基本卡片 7ipso 或自建 | 宽版、上下组合、含复选框 |
| 复选框 | f0TO6/zEXWw/Ee6NI | 勾选色 #0A59F7 |

**设计规范**：圆角 radius-sm 4px、radius-md 6px、radius-lg 8px；主文字 #191919、次文字 #595959、勾选 #0A59F7；间距 4 的倍数。

---

## 4. 实现顺序建议

1. 搭整体框架：顶栏（48px）+ 左侧导航（192px）+ 右侧角色树（192px）+ 主内容区
2. 主内容区：头信息 + 页签（权限/成员）+ 搜索框
3. 权限页签内容：左侧服务列表 + 右侧服务卡片（Req、Pipeline 等，每卡片含全选+任务权限勾选）
4. 细节：勾选框色 #0A59F7、文字色 #191919/#595959、圆角、间距
