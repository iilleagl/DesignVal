# 设置-基本信息 — 设计需求文档

本文档描述"设置-基本信息"页面的详细设计需求，用于指导 Pencil 设计工具
基于组件库 `designsystem/devUI/components/devUI2.pen` 进行设计实现。

---

## 1. 整体布局与视觉风格

**布局结构**：经典三栏——顶栏（全宽 × 48px 固定高度）+ 左侧手风琴导航栏（宽度 192px，满高）+ 右侧主内容区（剩余宽度，满高）。

**视觉风格**：
- 色调：明亮浅色底，白色内容卡片。主色 #0A59F7，危险色 #E02128（红色），主要文字色 #191919，次要文字色 #595959
- 组件特征：华为云 CodeArts 设计规范，统一圆角（4px/8px），轻阴影卡片，清晰的分区线
- 文字：主标题 16px/#191919，小标题 14px/#191919，正文 14px/#595959，辅助文字 12px/#595959
- 背景：页面整体背景 #F3F6F8，内容区域卡片白色 #FFFFFF

**画布尺寸**：1920 × 1080

## 2. 核心区域详解

- 深色背景（组件默认样式），水平排列
- 左侧：Logo/品牌区域 + 导航菜单
- 右侧：图标按钮组 + 用户头像区域

**内容物料**：

| 元素 | 类型 | 具体内容 |
|------|------|----------|
| Logo/品牌文字 | 文本 | CodeArts |
| 菜单项1 | 文本 | 首页 |
| 菜单项2 | 文本 | 项目 |
| 菜单项3 | 文本 | 服务 |
| 菜单项4 | 文本 | 更多 |
| 区域 | 文本 | 华北-北京四 |

---

### B. 左侧手风琴导航栏

- 页面左侧，宽度 192px，高度占满剩余空间（1080 - 48 = 1032px）
- 白色背景，右侧有 1px 分割线
- 顶部显示标题"设置"
- 包含搜索框、多个一级菜单（手风琴展开/收起）、二级菜单项
- 当前状态：「通用配置」一级菜单展开，「基本信息」二级菜单高亮（hover 状态）

**内容物料**：

| 元素 | 类型 | 具体内容 |
|------|------|----------|
| 导航标题 | 文本 | 设置 |
| 搜索框 | 控件 | 搜索框（占位文字：搜索） |
| 一级菜单1 | 文本+图标 | 通用配置（展开状态） |
| 　├ 二级菜单1-1 | 文本 | 基本信息（当前 hover 高亮） |

| 一级菜单2 | 文本+图标 | 安全与认证（收起状态） |
| 一级菜单3 | 文本+图标 | 成员管理（收起状态） |
| 一级菜单4 | 文本+图标 | 服务扩展（收起状态） |

**还原要点**：使用侧边栏手风琴组件 `K0pVg`，宽度 192px。在 descendants 中填齐标题"设置"、搜索框、所有一级/二级菜单文案。「通用配置」展开，「基本信息」项为 hover 高亮态。

### D. 组织信息区域

- 位于标题下方，白色卡片容器，圆角 8px，内边距 24px
- 卡片标题"组织信息"，字号 16px，字重 600，颜色 #191919
- 内容按垂直排列，分为：图标设置 + 表单项列表 + 操作按钮

- 编辑图标时展示浮层/内联区域，包含：
  - 可配图标颜色选项（色块：#0A59F7 蓝、#2CA01C 绿、#E37D29 橙、#E02128 红、#7B68EE 紫、#595959 灰）
  - 预制图标可选列表（多个简约图标供用户选择）

|------|------|----------|
| 区域标题 | 文本 | 组织信息 |
| 图标预览 | 图标 | 蓝底白字"C"，64×64px 圆角正方形 |
| 编辑图标链接 | 文本 | 编辑图标 |
| 图标颜色标签 | 文本 | 图标颜色 |
| 颜色选项1 | 色块 | #0A59F7（蓝，选中态） |
| 颜色选项2 | 色块 | #2CA01C（绿） |
| 颜色选项3 | 色块 | #E37D29（橙） |
| 颜色选项4 | 色块 | #E02128（红） |
| 颜色选项5 | 色块 | #7B68EE（紫） |
| 颜色选项6 | 色块 | #595959（灰） |
| 预制图标标签 | 文本 | 选择图标 |

**还原要点**：图标预览部分需手动绘制（圆角正方形 + 居中文字）。编辑图标的浮层/展开区域使用手动构建：色块为小圆形或圆角正方形排列，预制图标为图标网格。

## 3. 组件使用与还原要点

实现时优先从 `designsystem/devUI/components/devUI2.pen` 选用组件，
复制到设计稿后通过 ref 引用；引用后**必须**修改实例内的文案、
图标、占位内容以符合本页业务语义。

| 区域 | 建议组件 | 说明 |
|------|----------|------|
| 顶部导航栏 | 顶部导航栏 `Uexy2` | 高度 48px，填充 CodeArts 品牌名及菜单项 |
| 左侧导航 | 侧边栏-手风琴 `K0pVg` | 宽度 192px，含标题"设置"、搜索框、四个一级菜单 |
| 组织名称输入 | 表单组件-输入框 `Zk0Ev` | hasContent 状态，填入已有值 |
| 组织域名输入 | 表单组件-输入框 `Zk0Ev` | hasContent 状态，填入已有值 |
| 区域选择 | 表单组件-下拉选择 `Crm3Z` | 已选/关状态，显示已选区域 |
| 组织描述输入 | 表单组件-输入框 `Zk0Ev` | hasContent 状态，填入已有值 |
| 保存按钮 | 按钮-主要 `mYPZF`（md） | 文案"保存"，主色 #0A59F7 |
| 取消按钮 | 按钮-次要 `VKSqi`（md） | 文案"取消" |
| 删除按钮 | 按钮-主要 `mYPZF`（md） | 文案"删除组织"，改为红色 #E02128 |

**设计规范**：颜色使用 Design Token；字号 12/14/16/20/24/36 px；
间距为 4 的倍数；圆角与阴影与组件库一致。

## 4. 实现顺序建议

1. 先搭整体框架：画布 1920×1080 + 背景色 #F3F6F8
2. 插入顶部导航栏（全宽 × 48px），填充品牌名和菜单文案
3. 插入左侧手风琴导航栏（192px × 1032px），填充"设置"标题和所有菜单项
4. 创建主内容区容器（右侧剩余空间），添加内边距
5. 添加页面标题"基本信息"
6. 创建"组织信息"白色卡片，内置图标设置区域
7. 添加图标编辑的颜色选项和预制图标展示
8. 逐个添加四个表单输入项（组织名称、组织域名、区域、组织描述）
9. 添加保存和取消按钮组
10. 创建"删除组织"白色卡片，内置警告提示和删除按钮
11. 最后微调间距、颜色、圆角等细节
