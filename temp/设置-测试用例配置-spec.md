# 设置-测试用例配置 — 设计需求文档

本文档描述"设置-测试用例配置"页面的详细设计需求，用于指导 Pencil 设计工具
基于组件库 `designsystem/devUI/components/devUI2.pen` 进行设计实现。

---

## 1. 整体布局与视觉风格

**布局结构**：经典三栏——顶栏（全宽 × 48px 固定高度）+ 左侧手风琴导航栏（宽度 192px，满高）+ 右侧主内容区（剩余宽度，满高）。左侧导航包含"设置"标题和两级结构树菜单，右侧主内容区从上到下排列三张设置卡片。

**视觉风格**：
- 色调：浅色系，干净白色背景。页面整体背景 #F3F3F3，内容卡片白色 #FFFFFF，主色 #0A59F7
- 组件特征：华为云 CodeArts 设计规范，统一圆角（4px/8px），轻阴影卡片，清晰分区
- 文字：主标题 16px/#191919，小标题 14px/#191919，正文 14px/#595959，辅助文字 12px/#595959
- 背景：页面整体背景 #F3F3F3，卡片白色 #FFFFFF

**画布尺寸**：1920 × 1080

---

## 2. 核心区域详解

- 页面最顶部，全宽 1920px × 48px 固定高度
- 深色背景（组件默认样式），水平排列
- 左侧：Logo/品牌区域 + 导航菜单
- 右侧：图标按钮组 + 用户头像区域

**内容物料**：

| 元素 | 类型 | 具体内容 |
|------|------|----------|
| Logo/品牌文字 | 文本 | CodeArts |
| 菜单项1 | 文本 | 首页 |
| 菜单项2 | 文本 | 项目 |
| 菜单项3 | 文本 | 服务 |
| 菜单项4 | 文本 | 更多 |
| 区域 | 文本 | 华北-北京四 |

### B. 左侧手风琴导航栏

- 页面左侧，宽度 192px，高度占满剩余空间（1080 - 48 = 1032px）
- 白色背景，右侧有 1px 分割线
- 顶部显示标题"设置"
- 包含搜索框、多个一级菜单（手风琴展开/收起）、二级菜单项
- 当前状态：「测试管理」一级菜单展开，「用例配置」二级菜单高亮

**内容物料**：

| 元素 | 类型 | 具体内容 |
|------|------|----------|
| 导航标题 | 文本 | 设置 |
| 搜索框 | 控件 | 搜索框（占位文字：搜索） |
| 一级菜单1 | 文本+图标 | 通用配置（收起状态） |
| 一级菜单2 | 文本+图标 | 安全与认证（收起状态） |
| 一级菜单3 | 文本+图标 | 测试管理（展开状态） |
| 　├ 二级菜单3-1 | 文本 | 用例配置（当前高亮） |
| 　├ 二级菜单3-2 | 文本 | 执行配置 |
| 　└ 二级菜单3-3 | 文本 | 缺陷配置 |
| 一级菜单4 | 文本+图标 | 成员管理（收起状态） |

**还原要点**：使用侧边栏手风琴组件 `K0pVg`，宽度 192px。在 descendants 中填齐标题"设置"、搜索框、所有一级/二级菜单文案。「测试管理」展开，「用例配置」项为高亮态。

### C. 主内容区 — 页面标题

- 位于主内容区最顶部，距顶部与左侧各有适当内边距（24px）
- 显示页面标题"测试用例配置"，字号 20px，字重 600，颜色 #191919

**内容物料**：

| 元素 | 类型 | 具体内容 |
|------|------|----------|
| 页面标题 | 文本 | 测试用例配置 |

**还原要点**：手动创建文本节点，字号 20px，font-weight 600，颜色 #191919。

---

### D. 功能控制卡片（第1张卡片）

- 白色卡片容器，圆角 8px，内边距 24px，距左右边距 20px
- 卡片标题"功能控制"，字号 16px，字重 600，颜色 #191919
- 卡片内包含 3 个设置项，水平等分排列（三列）
- 每个设置项采用上下结构：
  - 上方：设置项标题（14px，#191919）
  - 下方：开关组件 + 说明文字（14px，#595959），水平排列

**内容物料**：

| 元素 | 类型 | 具体内容 |
|------|------|----------|
| 卡片标题 | 文本 | 功能控制 |
| 设置项1标题 | 文本 | 用例评审 |
| 设置项1开关 | 控件 | 开关（开启状态） |
| 设置项1说明 | 文本 | 开启后，测试用例需经过评审才能执行 |
| 设置项2标题 | 文本 | 用例基线 |
| 设置项2开关 | 控件 | 开关（开启状态） |
| 设置项2说明 | 文本 | 开启后，可创建和管理用例基线版本 |
| 设置项3标题 | 文本 | 自动同步结果 |
| 设置项3开关 | 控件 | 开关（关闭状态） |
| 设置项3说明 | 文本 | 开启后，执行结果将自动同步到关联需求 |

**还原要点**：卡片手动创建白色圆角矩形容器。每个设置项内的开关组件需手动绘制（圆角胶囊形状 + 滑块圆点），开启状态为蓝色 #0A59F7，关闭状态为灰色 #C9CDD4。三个设置项水平等分排列，间距 24px。

### E. 自定义选项卡片（第2张卡片）

- 白色卡片容器，圆角 8px，内边距 24px，距左右边距 20px

## 3. 组件使用与还原要点

实现时优先从 `designsystem/devUI/components/devUI2.pen` 选用组件，
复制到设计稿后通过 ref 引用；引用后**必须**修改实例内的文案、
图标、占位内容以符合本页业务语义。

| 区域 | 建议组件 | 说明 |
|------|----------|------|
| 顶部导航栏 | 顶部导航栏 `Uexy2` | 高度 48px，填充 CodeArts 品牌名及菜单项 |
| 左侧导航 | 侧边栏-手风琴 `K0pVg` | 宽度 192px，含标题"设置"、搜索框、四个一级菜单 |
| 功能控制-开关 | 手动绘制 | 圆角胶囊开关组件 + 滑块，蓝色开启/灰色关闭 |
| Key/Value输入框 | 文本输入框 `tdhOO` | hasContent 状态，显示已有值 |
| 字段输入框 | 文本输入框 `tdhOO` | hasContent 状态，显示已有值 |
| 表单标签 | 表单标题 `7iMMv` | 14px，#191919 |
| 添加按钮 | 按钮-图标文本按钮 `6LFBN` | md 尺寸，蓝色文字 |
| 保存按钮 | 按钮-主要 `mYPZF`（md） | 文案"保存"，主色 #0A59F7 |
| 取消按钮 | 按钮-次要 `VKSqi`（md） | 文案"取消" |

**设计规范**：颜色使用 Design Token；字号 12/14/16/20/24/36 px；
间距为 4 的倍数；圆角与阴影与组件库一致。

## 4. 实现顺序建议

1. 先搭整体框架：画布 1920×1080 + 背景色 #F3F3F3
2. 插入顶部导航栏（全宽 × 48px），填充品牌名和菜单文案
3. 插入左侧手风琴导航栏（192px × 1032px），填充"设置"标题和所有菜单项
4. 创建主内容区容器（右侧剩余空间），添加内边距
5. 添加页面标题"测试用例配置"
6. 创建"功能控制"白色卡片，内置 3 个开关设置项（水平三列）
7. 创建"自定义选项"白色卡片，内置 3 行 key/value 输入 + 按钮组
8. 创建"自定义字段"白色卡片，内置 3 行中英文字段输入 + 按钮组
9. 最后微调间距、颜色、圆角等细节
